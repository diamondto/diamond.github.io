Grid布局

CSSgrid布局

Flex 布局是轴线布局，只能指定”项目”针对轴线的位置，可以看作是一维布局。
Grid布局则是将容器划分成”行”和”列”，产生单元格，然后指定”项目所在”的单元。
可以看作是二维布局。Grid 布局远比 Flex 布局强大。

阮一峰CSS Grid 网格布局教程

抽取+精简重点如下：
1、grid布局是二维方式，直接设置父元素就行。
2、改变6个元素从列排列到行排列，可以设置grid-template-columns属性
3、fr是浮动属性，可以设置固定或者浮动。
4、colume-gap列间距，row-gap设置行间距。
5、grid-template-areas设置布局分配。
6、水平行轴，竖直是块轴。
7、水平使用justify-items对齐，垂直方向使用align-items。

CSS动画：
1、主要是应用transform，translate，transition，animation这4个属性，
2、transform主要定义元素的动作，比如旋转、倾斜、位移等，
3、translate是transform里面的属性，用于2D/3D位移。
4、后2个主要对动作进行描述，动画时间、速度曲线、次数等。
5、animation在不需要触发任何事件的情况下也可以显式的随着时间变化来改变元素css的属性值。
6、transition属性是一个简单的动画属性，需要触发一个事件， 非常简单容易用，是animation的简化版本，给普通做简单网页特效用。
